Project Description
Little Lemon Restaurant API

This project is a RESTful API for managing the menu and bookings of the Little Lemon restaurant. 
It is built using Django and Django REST framework, providing endpoints for creating, reading, 
updating, and deleting menu items and booking tables.

Features:
Menu Management: Add, view, update, and delete menu items.
Booking Management: Book tables and manage reservations.
Authentication: Secure endpoints using token-based authentication.
Endpoints:
/restaurant/menu-items/ - Manage menu items.
/restaurant/booking/ - Manage table bookings.

Technologies Used:
Django
Django REST framework
Djoser for authentication
This API allows the restaurant to efficiently manage its menu and bookings, 
ensuring a smooth operation and better customer experience.
